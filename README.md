## Requirements

- macOS 13.0+
- Xcode 14+
- Swift 5.7+

## Installation

1. Clone the repository
2. Open `ZenCalendar/Package.swift` in Xcode
3. Build and run

## Usage

- **Left-click** the menu bar icon to toggle the calendar
- **Right-click** for quit option
- **Click** the month/year header for quick date navigation
